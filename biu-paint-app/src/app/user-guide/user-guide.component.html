User Guide

This guide shortly explains functions of this programme and how to use them.

Pen Tool This tool allows free drawing on canvas with chosen size and color. Simply click on canvas and move your mouse, drawing stops once you lift your mouse button. Line Tool This tool allows drawing lines on canvas with chosen size and color. Click on canvas, move your mouse and lift mouse button. Eraser Tool This tool allows to erase mistakes. Tool uses currently active tool size, so be careful. Click on canvas and move your mouse, erasing stops once you lift your mouse button. Fill Tool This tool allows filling area with active color. Click on canvas and all pixels within closed area will be colored. Color Picker Tool This tool allows to change tool color. You can select new color by clicking on canvas. Square Tool This tool allows drawing squares on canvas with chosen size and color. Click on canvas drag you mouse and lift button. Circle Tool This tool allows drawing circles on canvas with chosen size and color. To be more precise it allows drawing ellipses. The place of mouse click is middle. Click on canvas and drag you mouse outwards of center and lift button when you are finnished. Triangle Tool This tool allows drawing triangles on canvas with chosen size and color. Click on canvas drag you mouse and lift button. Clear All This tool allows erasing everything from canvas returning to starting point. To use, select tool and click anywhere on canvas. Selection Tool This tool allows selecting area and performing some actions. To select area click on canvas, drag and lift your mouse button. Move Selection Use your arrow keys ← ↑ → ↓ to move selected area around canvas. Copy Selection By pressing Ctrl + C you can save selected area into programme's memory. You can have only one copy at a time. Paste Selection Once you created a copy you can paste it into canvas by pressing Ctrl + V. Confirm Changes Once your selected area is in desired position press Enter to apply changes. Cancel Changes If you don't like changes you can revoke (if not confirmed before) by pressing Escape / ESC button Text Tool This tool allows to insert Text onto your canvas. This tool uses currently active tool size as well as color. When you've entered your text, just click on canvas. Text will be drawn starting from click position and going upwards and right. Tool Color Changes currently active color. Tool Size Changes currently active tool size. Can't be less than 1. Canvas Size Changes canvas size to a new one. Everything on canvas will be scaled to new size. Import This tool allows adding your own picture. Export Allows to save your work on your disk.
